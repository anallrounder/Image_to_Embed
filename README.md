#### 노트 정리를 위한 이미지 저장 (중단함)
```
노트 어플을 사용하면서 불편했던 점은 
이미지를 그대로 업로드하면 컴퓨터에 이미 저장된 이미지 파일 외에도 
노트를 생성하면서 그만큼의 용량을 또 차지하게 된다는 점이었다.

그래서 에버노트를 사용하면서 점점 이미지 업로드는 하지 않고
글만 업로드해왔는데 이게 사실상 엄청게 답답한 부분이었다.
(현재는 에버노트를 거의 사용하지 않아서 임베드 기능이 제공되는지 모르겠다. 
하지만 예전엔 아마 있었어도 몰라서 사용못했을 듯)

그런데 현재 사용하는 노션에서는 Embed 기능이 제공되어 꼭 이 기능을 사용해야겠다고 생각했었다.
 
일단 깃허브에 업로드하는것이 쉬워보여서 일단 몇개를 올려보았다.
업로드 자체가 간단하기는 하지만 관리하기는 어려운 느낌이 들었다. 
생각보다 벌써 이미지 파일이 많아서 방금 올린 이미지를 찾는것도 쉽지 않아졌다.
게다가 공간을 얼만큼까지 사용할 수 있는지 모르겠어서 고민스럽다. 
아마 장기적으로는 구글드라이브가 나을 것 같아서 일단 업로드를 중단했다.
좀 더 알아봐야겠다.

201212-201224 기간동안 만든 이미지만 이곳에 올라와있고, 당분간 업데이트를 중단합니다.
```
