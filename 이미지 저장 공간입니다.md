#### 1. 노트 정리를 위한 이미지 저장 공간입니다.
```
예전부터 노트 어플을 사용하면서 느낀점은 
이미지를 그대로 업로드하면 컴퓨터에 이미 저장된 이미지 파일 외에도 
노트를 생성하면서 이미지 파일의 용량을 또 차지하게 된다는 점이었다.
그래서 에버노트를 사용하면서 점점 이미지 업로드는 하지 못하고 오로지 글만 업로드해왔는데 사실상 엄청게 답답한 부분이었다.
(현재는 에버노트를 거의 사용하지 않아서 임베드 기능이 제공되는지 모르겠다. 하지만 예전엔 아마 있었어도 몰라서 사용못했을 듯)
현재 사용하는 노션에서는 Embed 기능이 제공되어 꼭 이 기능을 사용해야겠다고 생각했었다.
원래 처음 계획은 구글 드라이브를 이미지 저장에 사용하려는 것이었으나, 어차피 내가 만든 그림이고 공개할 그림이라면 
여기 올려도 큰 문제는 없을 것 같아서 일단 여기에 업로드 하려고 한다.
이미지들의 날짜와 이름이 더 명확했더라면 좋았겠지만 이미 너무 많은 이미지를 업로드 한 관계로 이부분은 일단 보류(사실상 포기)한다.
```

#### 2. 역시나 폴더구조로 만들 수 있으면 좋겠다.
```
생각보다 이미지 파일이 너무 많아서 1달만에 업로드 하자마자 방금 올린 이미지를 찾는것도 쉽지 않다는 느낌이다.
앞으로 4-5개월간 얼마나 많은 이미지를 만들어 내게 될지 모르겠으나 트리구조로 폴더 정리를 하고싶다는 강렬한 생각이든다.
앞으로 이미지 저장시에 날짜를 앞에 쓰는 방법을 고려해야겠다.
```
