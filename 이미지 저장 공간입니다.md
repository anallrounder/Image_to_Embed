#### 노트 정리를 위한 이미지 저장을 하려고 했었다.
```
예전부터 노트 어플을 사용하면서 느낀점은 
이미지를 그대로 업로드하면 컴퓨터에 이미 저장된 이미지 파일 외에도 
노트를 생성하면서 이미지 파일의 용량을 또 차지하게 된다는 점이었다.
그래서 에버노트를 사용하면서 점점 이미지 업로드는 하지 못하고 
오로지 글만 업로드해왔는데 사실상 엄청게 답답한 부분이었다.
(현재는 에버노트를 거의 사용하지 않아서 임베드 기능이 제공되는지 모르겠다. 
하지만 예전엔 아마 있었어도 몰라서 사용못했을 듯)
현재 사용하는 노션에서는 Embed 기능이 제공되어 꼭 이 기능을 사용해야겠다고 생각했었다.

원래 처음 계획은 구글 드라이브를 이미지 저장에 사용하려는 것이었으나, 
일단 깃허브에 업로드하는것이 쉬워보여서 일단 몇개를 올려보았다.
간단하기는 하지만 관리하기는 어려운 느낌이 들었다. 
게다가 공간을 얼만큼까지 사용할 수 있는지 모르겠어서 고민스럽다. 
아마 장기적으로는 구글 드라이브를 이용하게 될 것 같다. 
그리고 이미 어느정도 많은 이미지를 업로드 해서 이름으로 찾기가 어려워졌다. 
이름 앞부분에 날짜를 넣어야하나 고민되는 부분. 
그런데 어차피 여기보다 구글드라이브가 나을 것 같아서 
일단 이미지는 이사를 갈 예정이라서 수정이나 재 업로드 등의 관리는 보류(사실상 포기)한다.

생각보다 이미지 파일이 너무 많아서 이미 지금도 업로드 하자마자 방금 올린 이미지를 찾는것도 쉽지 않다는 느낌이다.
이미지는 구글드라이브로 옮기는게 장기적으로 좋을것같다.
201212-201224 기간동안의 이미지만 이곳에 올라와 있으므로 당분간 업데이트를 중단합니다.
```
